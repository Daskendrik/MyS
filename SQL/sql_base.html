У нас будут следующие колонки: Row_id, Family, LastName, First_name, Middle_name, Telephone, Mark, Model. Название таблицы Telephone. Поля, которые не должны быть пустыми должны маркироваться NOT NULL. В таблицах так же должен присутствовать ключ, в нашей таблице это row_id, и чтобы самим не запоминать и не придумывать этот ключ поручим это таблицы и подпишем AUTO_INCREMENT.

CREATE TABLE Telephone (
Row_id int AUTO_INCREMENT,
Family int NOT NULL,
LastName varchar(20) NOT NULL,
First_name varchar(20) NOT NULL,
Middle_name varchar(20),
Telephone int NOT NULL,
Mark varchar(20),
Model varchar(20),
PRIMARY KEY (Row_id)
