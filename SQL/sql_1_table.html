Первый запрос

Практически все запросы начинаются с Select. То есть, можно сказать, Выбираю. Самый простой запрос будет выглядеть так:

Select * from Telephone  -- дословно Выбрать все из таблицы Telephone (* = все)

Или

Select Telephone .* from Telephone 

Или

Select a.* from Telephone  a

Первый вариант подходит для самых простых запросов, но дальше такой способ может не сработать и запутать. Второй вариант содержит много символов, не то, чтобы это плохо, но лучше использовать третий вариант. Таблицу мы как бы переименовываем на а для упрощения запроса. Лучше приучаться писать сразу третьим вариантом. Плюс ко всему, запросы можно делать более красиво, не в 1 строчку. Руками красоту конечно делать не стоит, обычно такой функционал есть в программе, в который вы пишите запросы 

select a.*

from Telephone a

Нам будет показана вся таблица, без каких либо сортировок и тд и тп.
