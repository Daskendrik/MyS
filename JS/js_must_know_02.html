  1. Типы данных

    • Число: все числа и NaN
    • Строки: 'пишется в кавычках'
    • Boolean: true или false (0, null,undefined, NaN и пустая строка считается false)
    • Object: {}
    • Null: если ничего, но определено. Только Null
    • Undefined: undefined

  2. Операции

    • Str + str = str ('a' + 'b' = 'ab')
    • Str + nub = str ('1' + 2 = '12')
    • Num + num = num (1 + 2 = 3)
    • Bool + num = num (true + 1 = 2)
    • Bool + str = str (true + 'a' = truea)
    • Num - num = num (2 - 1 = 1)
    • Num * num = num (2 * 1 = 2)
    • Num / num = num (2 / 1 = 2)
    • = - присвоение
    • (i++) постфиксная ++ или -- - инкримент/дикремент, сначала какой то результат, потомизменение
    • (++i) префиксная ++ или -- сразу по приоритету получаем результат
    • += или -= или *= или /= - сначала действие, потом присвоение
    • Num % num = num (5 % 2 = 1) - возвращает остаток
    • Равенство ==
    • Неравестнво !=
    • Строгое равенство, проверяется и тип ===
    • Строгое неравенство, проверяется и тип !==
    • Приоритет как в математике. Подробнее

  4. Операторы циклы

    • for(var i = 0; условие; i++ (шаг)){} - цикл с переменной внутри условия
    • continue - используется для выхода из дочернего оператора в родителя
    • breake - завершает дочерний и родительский оператор
    • while(условие) {действие} - делай пока условие совпадает, остановится, как только условие не совпадет, может ниразу не выполнить действие
    • do {действие} while (условие) - делай это пока условие выполняется. Отличается тем, что хотя бы раз действие будет выполнено.

  5. Функции

    • function nameOwn(параметр) {действие} - такое создание функции видно по всему файлу
    • var/let Name = function(параметр) {действие} - такое создание функции видно только после ее определения
    • print a или print(a) - вернет тип данных
    • console.log() - выводит () в консоль

  6. Методы строк

    • str.leght - вернет кол-во символов
    • str.toUpperCase() - вернет строку, где буквы с верхним регистром
    • str.toLowerCase() - вернет строку, где буквы с нижним регистром
    • str.charAt(позиция) - возвращает символ на позиции
    • str.indexOf('параметр') - возвращает позицию параметра в строке (если параметра нет, вернет -1)
    • str.substr(начало отчета, сколько символов) - вернет вырезку из строки
    • str.substring(начало отчета, сколько оставить от строки) - вернет строки, которые вместились во второй параметр. Если второй параметр пустой, то вырезаются символы первого параметра

  7. Методы чисел

    • 20е3 = 20000, е = 10, 3 - сколько раз на 10 умножить
    • num.toString() - переводит число в строку
    • num.toFixed(x) -округялет в большую сторону на x знаков после запятой и переводит в строку
    • +a - переводит a в число
    • parseFloat(a) - переводит строку a в число, не убирая дробную часть
    • parseInt(a) - переводит строку a в число убирая дробную часть
    • isNaN(a) - true если число не валидное, иначе false
    • isFinite(a) - false, если число бесконечно, true - если конечно

  8. Массив и методы

    • var m = ['param1', param2] - объявление массива
    • arr[x] - берем значение , которая находится на позиции x
    • arr.lenght - длинна массива
    • arr.push('value') - добавляет новый элемент в конец массива со значением value
    • arr.unshift('value') - добавляет новый элемент в начало массива со значением value
    • arr.pop() - удаляет последний элемент в массиве и возвращает его
    • arr.shift()- удаляет и возвращает первый элемент
    • arr.unshift('value') - добавляет элемент в начало массива
    • arr.indexOf('') - позволяет определить, есть ли элемент со значением 'value' в массиве (-1 нет, x - индекс этого значения)

    • someObject.key - вернет значение ключа

  10. DOM

    • DOM - это по сути код страницы
    • document. - самый глобальный объект, имеет свои методы
    • старый метод, но стоит использовать именно его let a = document.getElementByID('id') - получаем доступ к определенному объекту по ID(единственный уникальный)
    • старый метод let a = document.getElementsByClassName('class') - получаем доступ к определенному объекту по классу(дублируется бес-но)
    • старый метод let a = document.getElementsByTagName('tag') - получаем доступ к определенному объекту по тегу
    • let a = document.querySelector('#ID') - получаем доступ к определенному объекту по ID. Возвращает только 1 запись
    • let a = document.querySelectorAll('.class') - получаем доступ к определенному объекту по классу. Возвращает все записи
    • let a = document.querySelector('tag') - получаем доступ к определенному объекту по тегу
    • let a = document.querySelector('div#D5 > ul') - получаем доступ дочкам
    • a.innerHTML -получаем доступ к html содержимому, можем менять
    • a.textContent - получаем содежржимое, без html, можем менять
    • a.getAttribute('атрибут') - получаем значение 'атрибут', и записывать в переменную
    • a.setAttribute('атрибут', 'значение') - задает 'значение' для 'атрибут'
    • a.attribetes - возвращает объект, включающий все атрибуты
    • a.classList - углубляемся в классы объекта html
    • a.classList.add('класс') - добавляем 'класс'
    • a.classList.remove('класс') - удаляет 'класс'
    • a.classList.contains('класс') - проверяет наличие 'класс'
    • a.style - дает доступ к стилям

  11. DOM. События

    • a.addEventListener('название события(например click)', функция, которая будет выполняться(без ()),true/false(по умолчанию фолс))
    • 3 параметр в addEventListener определяет порядок
    • В параметр функции (привызове addEventListener) можно передавать event - объект, который включает в себя подробности события (на что нажали, какую кнопку зажимали, координаты клика)
    • event.target - название объекта, на который
    • event.stopPropagation() - останавливает событие на элементе, на котором произошел клик
    • event.preventDefault() - отменяет дефолтный клик (например у a переход по ссылке)
    • Не стоит добавлять событие каждому элементу, если у него есть обертка, лучше работать через нее
    • События мыши
