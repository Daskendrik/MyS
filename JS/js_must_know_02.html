Тут я хочу написать то, что стоит помнить и повторять.

  1. Функции смешанные

    • print a или print(a) - вернет тип данных

  2. Базовые операции

    • Str + str = str ('a' + 'b' = 'ab')
    • Str + nub = str ('1' + 2 = '12')
    • Num + num = num (1 + 2 = 3)
    • Bool + num = num (true + 1 = 2)
    • Bool + str = str (true + 'a' = truea)
    • Num - num = num (2 - 1 = 1)
    • Num * num = num (2 * 1 = 2)
    • Num / num = num (2 / 1 = 2)
    • = - присвоение
    • (i++) постфиксная ++ или -- - инкримент/дикремент, сначала какой то результат, потомизменение
    • (++i) префиксная ++ или -- сразу по приоритету получаем результат
    • += или -= или *= или /= - сначала действие, потом присвоение
    • Num % num = num (5 % 2 = 1) - возвращает остаток
    • Равенство ==
    • Неравестнво !=
    • Строгое равенство, проверяется и тип ===
    • Строгое неравенство, проверяется и тип !==
    • Приоритет как в математике. Подробнее
    • Логический опреатор || ИЛИ
    • Логический опреатор && И
    • Логический опреатор ! НЕ
