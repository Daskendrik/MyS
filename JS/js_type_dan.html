Методы разных типов данных

Числа

Способы записи числа - e в числе это 10^x. То есть, 1е9 это 1 * 10^9 = 1 000 000 000 . Примеры:

  • 7.3e9 = 7 300 000 000 (один ноль ушел на точку)
  • 1.23е6 = 1 230 000
  • 1е-6 = 0,000 001

    • toString(base) - метод, который возвращает строковое представление числа в систему base, по умолчанию 10 (2,8,16 ... 36 -тиричную систему). Примеры (let num = 255;):
  • alert( num.toString(16) ); // ff
  • alert( num.toString(2) ); // 11111111

    • Округления - существет много способов округления. Примеры (let num = 3.6;): (МОЖНО БУДЕТ НАПИСАТЬ ПРОГУ!)
  • Math.floor - Округление в меньшую сторону
    Math.floor(num) - 3
  • Math.ceil- Округление в большую сторону
    Math.ceil(num) - 4
  • Math.round - Округление до ближайшего целого
    Math.round(num) - 4
  • Math.trunc - Производит удаление дробной части без округления
    Math.trunc(num) - 3
  • toFixed(n) - округляет число до n знаков после запятой и возвращает строковое представление результата
    num.toFixed(1) - 3.6

    • isNaN(value) - преобразует значение в число и проверяет является ли оно NaN. Пример:
  • alert( isNaN("str") ); // true (является ли "str" NaN?)

    • isFinite(value) - преобразует аргумент в число и возвращает true, если оно является обычным числом, т.е. не NaN/Infinity/-Infinity. Пример:
  • alert( isFinite("15") ); // true

    • + или Number() - преобразует к числу, только если строка является в точности числом(Единственное исключение — это пробелы в начале строки и в конце, они игнорируются). Например:
  • alert(+"100") // 100

    • parseInt - читает число из строки (если число стоит на 1 месте), возвращается целое число. Имеет необязательный второй параметр. Он определяет систему счисления, таким образом parseInt может также читать строки с шестнадцатеричными числами, двоичными числами и т.д. Пример:
  • alert( parseInt('12.3') ); // 12, вернётся только целая часть
  • alert( parseInt('0xff', 16) ); // 255

    • parseFloat - читает число из строки (если число стоит на 1 месте), возвращает число с плавающей точкой. Пример:
  • alert( parseFloat('12.5em') ); // 12.5

    • Math.random() - возвращает псевдослучайное число в диапазоне от 0 (включительно) до 1 (но не включая 1). Пример:
  • alert( Math.random() ); // 0.1234567894322

    • Math.max(a, b, c...) / Math.min(a, b, c...) - возвращает наибольшее/наименьшее число из перечисленных аргументов. Пример:
  • alert( Math.max(3, 5, -10, 0, 1) ); // 5

    • Math.pow(n, power) - возвращает число n, возведённое в степень power. Пример:
  • alert( Math.pow(2, 10) ); // 2 в степени 10 = 1024

    • Строки

    Инфа
    Кавычки - создают строковые данные. Примеры:

  • let single = 'single-quoted'; //одинарные кавычки
  • let double = "double-quoted"; //двойные кавычки
  • let backticks = `backticks`; //обратные кавычки
    • Обрытные кавычки - в такую строку мы сможем вставлять произвольные выражения, обернув их в ${…}. Ещё одно преимущество обратных кавычек — они могут занимать более одной строки. Пример:
  • alert(`1 + 2 = ${sum(1, 2)}.`); // 1 + 2 = 3. sum(a, b) - функция суммы
    • Спец-символы используются для переноса строки или табуляции, перечень всех:
  • \n - Перевод строки
  • \r - Возврат каретки: самостоятельно не используется. В текстовых файлах Windows для перевода строки используется комбинация символов \r\n.
  • \' \" - Кавычки
  • \\ - Обратный слеш
  • \t - Знак табуляции
  • \xXX - Символ с шестнадцатеричным юникодным кодом XX, например, '\x7A' — то же самое, что 'z'
  • \uXXXX - Символ в кодировке UTF-16 с шестнадцатеричным кодом XXXX, например, \u00A9 — юникодное представление знака копирайта, ©. Код должен состоять ровно из 4 шестнадцатеричных цифр.
    • Длина строки - Свойство length содержит длину строки. Пример:
  • alert( `My\n`.length ); // 3
  • length — это свойство
    • Доступ к символам - Получить символ, который занимает позицию pos, можно с помощью квадратных скобок: [pos]. Также можно использовать метод charAt: str.charAt(pos). Первый символ занимает нулевую позицию. Примеры (let str = `Hello`;):
  • alert( str[0] ); // H наиболее актуальный вариант
  • alert( str.charAt(0) );
  • alert( str[str.length - 1] ); // o получаем последний символ
  • Также можно перебрать строку посимвольно, используя for..of

    for (let char of "Hello") {
    alert(char); // H,e,l,l,o (char — сначала "H", потом "e", потом "l" и т. д.)
    }

    • Строки неизменяемы - Содержимое строки в JavaScript нельзя изменить. Нельзя взять символ посередине и заменить его. Как только строка создана — она такая навсегда.
    Изменение регистра - регистр (заглавная или маленькая) меняется 2 методами toLowerCase() и toUpperCase(). Пример:
  • alert( 'Interface'.toUpperCase() ); // INTERFACE
  • alert( 'Interface'.toLowerCase() ); // interface
  • alert( 'Interface'[0].toLowerCase() ); // 'i' Если мы захотим перевести в нижний регистр какой-то конкретный символ
    • Поиск подстроки.Существует несколько способов поиска подстроки: str.indexOf(substr, pos), str.lastIndexOf(substr, position) . Пример (let str = 'Widget with id')
  • str.indexOf
    alert( str.indexOf('Widget') ); // 0, потому что подстрока 'Widget' найдена в начале
    alert( str.indexOf("id") ); // 1, подстрока "id" найдена на позиции 1 (..idget with id)
    alert( str.indexOf('id', 2) ) // 12 (второй параметр задает с какого символа начать)
  • str.lastIndexOf(substr, position) - ищет с конца строки к её началу
  • str.includes(substr, pos) - возвращает true, если в строке str есть подстрока substr, либо false, если нет.
    alert( "Widget with id".includes("Widget") ); // true
    alert( "Midget".includes("id", 3) ); // false, поиск начат с позиции 3
  • str.startsWith и str.endsWith - проверяют, соответственно, начинается ли и заканчивается ли строка определённой строкой
    alert( "Widget".startsWith("Wid") ); // true, "Wid" — начало "Widget"
    alert( "Widget".endsWith("get") ); // true, "get" — окончание "Widget"
    • Получение подстроки есть 3 метода для получения подстроки: substring, substr и slice.
  • str.slice(start [, end]) - Возвращает часть строки от start до (не включая) end. (let str = "stringify";)
    alert( str.slice(0, 5) ); // 'strin', символы от 0 до 5 (не включая 5)
    alert( str.slice(0, 1) ); // 's', от 0 до 1, не включая 1, т. е. только один символ на позиции 0
    alert( str.slice(2) ); // ringify, с позиции 2 и до конца
    alert( str.slice(-4, -1) ); // gif начинаем с позиции 4 справа, а заканчиваем на позиции 1 справа
  • str.substring(start [, end]) - Возвращает часть строки между start и end. (let str = "stringify";)
    alert( str.substring(2, 6) ); // "ring"
    alert( str.substring(6, 2) ); // "ring"
  • str.substr(start [, length]) - Возвращает часть строки от start длины length.
    alert( str.substr(2, 4) ); // ring, получаем 4 символа, начиная с позиции 2
    • Сравнение строк (значения строк в цифрах)

    Символы сравниваются по их кодам.

  • str.codePointAt(pos) - Возвращает код для символа, находящегося на позиции pos:
    alert( "z".codePointAt(0) ); // 122
    alert( "Z".codePointAt(0) ); // 90
  • String.fromCodePoint(code) - Создаёт символ по его коду code
    alert( String.fromCodePoint(90) ); // Z
  • str.localeCompare(str2) - возвращает число, которое показывает, какая строка больше в соответствии с правилами языка (отрицательно число - меньше, положительное - больше, 0 равны)
    alert( 'Österreich'.localeCompare('Zealand') ); // -1
    • Дополнительно
  • str.trim() - убирает пробелы в начале и конце строки.
  • str.repeat(n) — повторяет строку n раз.

    • Методы для всего

    Object.is - сравнивает значения примерно как ===, но более надёжен. Пример:

  • Object.is(2, 2) //true
