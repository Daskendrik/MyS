Массив

Массив это тоже объект, но немного другого типа. Это упорядочная коллекция данных. Объявляется массив двумя способами:

let arr = new Array();
let arr = [];

Например

let fruits = ["Яблоко", "Апельсин", "Слива"];
alert( fruits[0] ); // Яблоко
alert( fruits[1] ); // Апельсин показали
fruits[2] = 'Груша'; // теперь ["Яблоко", "Апельсин", "Груша"]
fruits[3] = 'Лимон'; // теперь ["Яблоко", "Апельсин", "Груша", "Лимон"]
alert( fruits.length ); // 4
alert( fruits ); // Яблоко, Апельсин, Груша, Лимон
let arr = [ 'Яблоко', { name: 'Джон' }, true, function() { alert('привет'); } ];

Нумерация начинается с 0. чтобы обратится к необходимому параметру, его порядлковый номер пишем в []. Таким образом, мы можешь заменять, показывать, добавлять. Общее число элементов массива содержится в его свойстве length. Вывести массив целиком можно при помощи alert. В массиве могут храниться элементы любого типа.

Методы

  • push добавляет элемент в конец (fruits.push("Груша");)
  • pop удаляет последний элемент и возвращает его (alert( fruits.pop() ); // удаляем "Груша" и выводим его)
  • shift удаляет из массива первый элемент и возвращает его (alert( fruits.shift() ); // удаляем Яблоко и выводим его)
  • unshift добавляет элемент в начало массива
  • Методы push и unshift могут добавлять сразу несколько элементов

    • Для перебора всех элементов массива можно использовать следующую конструкцию:

    let fruits = ["Яблоко", "Апельсин", "Слива"];

    // проходит по значениям
    for (let fruit of fruits) {
      alert( fruit );
    }

    Чтобы удалить все элементы достаточно написать так: arr.length = 0;. Дело в том, что когда мы сокращаем (задаем длинную меньшую существующей), мы безвовратно удаляем все элементы, которые не вошли в диапазон.

    Многомерный масив

    Это массив, который в себе имеет другие массивы, иначе говоря, это матрица.

    let matrix = [
      [1, 2, 3],
      [4, 5, 6],
      [7, 8, 9]
    ];

    alert( matrix[1][1] ); // 5, центральный элемент
